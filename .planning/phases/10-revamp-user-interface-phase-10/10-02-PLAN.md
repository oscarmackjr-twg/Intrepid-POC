Replace Layout.tsx entirely. The new component must implement all of the following:

**Outer shell:**
```
<div class="flex flex-col min-h-screen">
  <StagingBanner />                         {/* full-width amber bar, sticky top */}
  <div class="flex flex-1">
    <aside>                                  {/* sidebar */}
    <main>                                   {/* content */}
  </div>
</div>
```

StagingBanner MUST be outside the flex row, rendered first so it spans the full width above both sidebar and content. Do not wrap it inside the aside or main.

**Sidebar (`<aside>`):**
- `w-60 min-h-full flex flex-col bg-white border-r border-gray-200` (240px = w-60)
- Fixed to the left — use `sticky top-0 h-screen` so it stays in view while content scrolls
- No drop shadow on the sidebar itself (conservative financial aesthetic)
- Three sections: logo header, nav list (flex-1 overflow-y-auto), user footer

**Sidebar logo header:**
```jsx
<div className="px-5 py-5 border-b border-gray-200">
  <img src={twgLogo} alt="TWG Global" className="h-8 w-auto" />
  <p className="mt-1 text-xs font-semibold tracking-widest text-[#1a3868] uppercase">
    Intrepid Loan Platform
  </p>
</div>
```

**Sidebar nav:**
Build the nav using three item types:
1. **Direct nav link** — `<Link to={path}>` with active state
2. **Group label** — `<span>` (not a Link), styled as a section header
3. **Child nav link** — indented `<Link to={path}>` with active state

Active state (apply when `location.pathname.startsWith(item.basePath)`):
```
border-l-4 border-[#1a3868] text-[#1a3868] font-semibold bg-gray-50
```
Inactive state:
```
border-l-4 border-transparent text-[#475569] hover:text-[#1a3868] hover:bg-gray-50
```

Group label style (non-clickable):
```
px-5 pt-4 pb-1 text-xs font-bold tracking-widest text-[#94a3b8] uppercase select-none
```

Child link indentation: add `pl-8` on top of the standard link padding.

Standard link padding: `px-5 py-2 text-sm flex items-center gap-2`
